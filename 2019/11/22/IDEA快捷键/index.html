IDEA快捷键

编辑

Ctrl+Space 基本代码完成（任何类的名称，方法或变量）

Ctrl+Shift+Space 智能代码完成（flters方法和变量（按预期类型）

Ctrl+Shift+Enter 完整语句

Ctrl+P 参数信息（在方法调用参数内）

Ctrl+Q 快速文档查找

Shift+F1 外部文档

Ctrl+ 鼠标 在代码摘要信息上

Ctrl+F1在插入符号处显示错误或警告的说明

Alt+Insert生成代码。。。（getter，setter，构造函数，hashCode/equals，toString）

Ctrl+O 覆盖方法

Ctrl+I 实现方法

Ctrl+Alt+T使 用…（如果..否则， 尝试..捕获，for，同步等）

Ctrl+/Comment/uncomment和line Comment

Ctrl+Shift+/Comment/uncomment和block Comment

Ctrl+W选择连续增加的代码块

Ctrl+Shift+W将当前所选内容减少到以前的状态

Alt+Q上下文信息

Alt+Enter显示意图动作和快速fxes

Ctrl+Alt+L重新格式化代码

Ctrl+Alt+O优化导入

Ctrl+Alt+I自动缩进行

制表符/移位+制表符缩进/取消缩进选定行

Ctrl + X ， Shift + 删除剪切当前行或选定块到剪贴板

Ctrl + C ， Ctrl + 将复制当前行或选定块插入剪贴板

Ctrl + V ， Shift + 从剪贴板插入粘贴

按Ctrl+Shift+V从最近的餐厅粘贴。。。

Ctrl + D复制当前行或选定块

Ctrl + Y删除插入符号处的行

Ctrl + Shift+J智能线路连接

Ctrl + 输入智能线拆分

Shift+Enter开始新行

Ctrl+Shift+U切换插入符号或选定块处的单词大小写

Ctrl+Shift+]/[选择直到代码块结束/开始

Ctrl+Delete/BackspaceDelete结束/开始

Ctrl+NumPad+/-展开/折叠代码块

Ctrl+Shift+NumPad+Expand all组合键

Ctrl+Shift+NumPad-全部折叠

Ctrl+F4关闭活动编辑器选项卡

使用情况搜索

Ctrl+Shift+F7突出显示fle中的用法

Ctrl+Alt+F7显示用法

导航

Ctrl+N转到类

Ctrl+Shift+N转到fle

Ctrl+Alt+Shift+N转到符号

Alt+右/左转到下一个/上一个编辑器选项卡

F12返回上一个工具窗口

Esc转到编辑器（从工具窗口）

Shift+Esc隐藏活动窗口或最后一个活动窗口

Ctrl + Shift + F4关闭活动运行/消息/fnd/。。。制表符

Ctrl + G转到行

Ctrl+E最近的fles弹出式菜单

Ctrl + Alt+左/右导航后退/前进

Ctrl + Shift + Backspace导航到最后一个编辑位置

Alt+F1选择任意视图中的当前fle或符号

Ctrl+B，Ctrl+单击“转到声明”

Ctrl+Alt+B转到实现

Ctrl+Shift+I打开快速定义查找

Ctrl+Shift+B转到类型声明

Ctrl+U转到超级方法/超级类

Alt+向上/向下转到上一个/下一个方法

Ctrl+]/[移动到代码块结束/开始

Ctrl+F12文件结构弹出菜单

Ctrl+H类型层次结构

Ctrl+Shift+H方法层次结构

Ctrl+Alt+H调用层次结构

F2/Shift+F2下一个/上一个突出显示的错误

F4/Ctrl+输入编辑源/查看源

Alt+Home Show导航栏

F11切换书签

Ctrl+F11使用助记键切换书签

Ctrl+#[0-9]转到编号书签

Shift+F11显示书签

搜索/替换

Double Shift 搜索

Ctrl+F查找

F3/Shift+F3查找下一个/查找上一个

Ctrl+R替换

Ctrl+Shift+F在路径中查找

在路径中按Ctrl+Shift+R替换

按Ctrl+Shift+S结构搜索（仅限最终版本）

Ctrl+Shift+M结构替换（仅限最终版本）

重构

F6移动

Shift+F6重命名

Ctrl+F6更改签名

Ctrl+Alt+F提取字段

Ctrl+Alt+C提取常量

调试

F9恢复程序
